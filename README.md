# 敏捷笔记
## 什么是敏捷？
敏捷是一种通过创造变化和响应变化在不确定和混乱的环境中取得成功的能力。

## 什么敏捷软件开发？
敏捷软件开发是基于敏捷宣言定义的价值观和原则的一系列方法和实践的总称。自组织、跨职能团队运用适合他们自身环境的实践进行演进得出解决方案。
